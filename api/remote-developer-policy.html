--- layout: developerpolicy title: Tempest API Remote Data Access Policy ---

This policy, which is subject to change, applies to remote data access. This policy does not apply to local data access.

Summary:

All data (qualilty-controlled observations, augmentation with additional network data, metadata & proprietary forecasts) is available to station owners from stations that they own (public or private) via any application for personal use. No data (observations, forecast or metadata) is available from private stations via any application. Metadata from public stations is available to any user via any application. Observation and forecast data from public stations may be available to any user via licensed commercial applications (please contact us for details).

Definitions:

Device a physical device (Hub, AIR, SKY)
Station a collection of devices
Application software used to access data
User individual requesting data
Owner user who owns a station
Public station owner has chosen to share publicly
Private station owner has chosen to NOT share publicly
Observation data observed weather parameters from a station (e.g. T, RH, UV, etc.)
Forecast data predicted weather parameters from a station (e.g. T, RH, UV, etc.)
Metadata data about a station (e.g. name, location, elevation, etc.)

Personal

Available to any user - no special agreement required

  • access to metadata, observation & forecast data by owner of station
  • access to public metadata from ALL public stations by any user
  • no access to private data by any user
  • no access to observation or forecast data by any user
  • generous rate/volume limits (enough for personal use and small-scale apps)
  • limited to personal use
  • rebroadcast may be limited

Commercial

Partnership agreement required. Contact us.

  • access to metadata, observation & forecast data by owner of station
  • access to public metadata from ALL public stations by any user
  • no access to private data by any user
  • may include access to observation or forecast data by any user
  • may include higher rate/volume limits, if necessary
  • Authorization to use data in a commercial manner under terms specified in partnership agreement
  • Rebroadcast only according to terms specified in partnership agreement
